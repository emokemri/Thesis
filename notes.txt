mit miért használtam (alternatívákkal való összehasonlítás)
 - Python és Tkinter: Ha a projekt már Pythonban van megvalósítva, akkor a Tkinter egy beépített GUI toolkit a Pythonban, ami lehetővé teszi egyszerű és viszonylag könnyű módon grafikus felületet készíteni.
 - Qt: A Qt egy népszerű és teljes körű GUI keretrendszer, amely támogatja a Python nyelvet is (PyQt vagy PySide néven). A Qt lehetővé teszi professzionális és jól testreszabható felületek létrehozását.
 - Web alkalmazás: Ha a grafikus felületet webes alkalmazásként szeretnéd megvalósítani, akkor HTML, CSS és JavaScript technológiákat használhatsz. Ebben az esetben a Flask vagy Django keretrendszereket is bevonhatod a Python oldalon a háttérlogika kezeléséhez.
 - .NET WPF vagy WinForms: Ha a Windows platformra célozol, akkor a .NET keretrendszer részeként elérhető Windows Presentation Foundation (WPF) vagy Windows Forms (WinForms) keretrendszereket használhatod.
 - Flutter vagy React Native: Ha mobilalkalmazást is tervezel a projekt részeként, akkor a Flutter (Dart programozási nyelv) vagy a React Native (JavaScript) lehet a megfelelő választás, mivel mindkettő lehetővé teszi az Android és iOS alkalmazások fejlesztését egy közös kódbázison.
